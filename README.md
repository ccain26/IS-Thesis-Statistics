### R Files Information
Three files - Spatial Testing Z.Rmd, Spatial Testing S.Rmd, Spatial Testing E.Rmd.
1. Spatial Testing Z.Rmd contains code for Z1, Z2, Z3.
2. Spatial Testing S.Rmd contains code for S1, S2, S3.
3. Spatial Testing E.Rmd contains code for E1 and E2.

### Data
1. .Rmd files need lots of external libraries.
2. Datasets must be downloaded for each .Rmd before running. You must make sure the .Rmd files have the right filepaths for each .csv.
